## Build and Test Instructions

Follow these steps to build the project and run tests:

1. **Clone the repository:**

   ```bash
   cd MTDS_lab2
   ```

2. **Install dependencies:**
   
   ```bash
   npm install
   ```

5. **Run tests:**

   ```bash
   npm test
   ```
